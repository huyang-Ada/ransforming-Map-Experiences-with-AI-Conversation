## 🔧 如何切换AI引擎

### 方法1：界面切换（推荐）

1. 打开Demo页面
2. 点击左侧「⚙️ API配置」按钮
3. 在「AI服务选择」下拉框中选择：
   - **本地规则引擎（免费）**
   - **通义千问 API**（默认）
   - **智谱 GLM API**
   - **自定义API**
4. 点击「保存AI配置」按钮

切换后立即生效，无需刷新页面！

---

### 方法2：修改代码切换

打开 [js/preset-config.js](js/preset-config.js)，修改第23行：

```javascript
// 默认AI服务配置（可切换）
aiConfig: {
    provider: 'qwen',  // 改为 'zhipu' 或 'local'
    apiKey: AI_PROVIDERS.qwen.apiKey,
    apiUrl: AI_PROVIDERS.qwen.apiUrl
}
```

**切换示例**：

**使用智谱GLM**：
```javascript
provider: 'zhipu',
apiKey: AI_PROVIDERS.zhipu.apiKey,
apiUrl: AI_PROVIDERS.zhipu.apiUrl
```

**使用本地引擎**：
```javascript
provider: 'local',
apiKey: null,
apiUrl: null
```

修改后刷新页面即可。

## ⚠️ 注意事项

### 1. 配额监控
云端AI有配额限制：
- 通义千问：100万tokens（免费）
- 智谱GLM：100万tokens（免费）
- 测试阶段建议先用本地引擎

### 2. 切换生效
界面切换立即生效，无需刷新
代码切换需要刷新页面

### 3. 失败回退
如果云端AI调用失败（配额用尽、网络问题），会自动回退到本地引擎
