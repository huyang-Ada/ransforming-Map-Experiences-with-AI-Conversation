# API申请指南

本项目需要以下API密钥才能完整运行。请按照本指南逐步申请。

---

## 一、腾讯位置服务API密钥申请

### 1. 注册账号

1. 访问 [腾讯位置服务官网](https://lbs.qq.com/)
2. 点击右上角「登录/注册」
3. 使用微信扫码或手机号注册账号

### 2. 创建应用

1. 登录后进入 [控制台](https://lbs.qq.com/console/mykey.html)
2. 点击「应用管理」 → 「我的应用」 → 「创建应用」
3. 填写应用信息：
   - **应用名称**: `AI智能地图助手`（或自定义）
   - **应用类型**: `Web端应用`
   - **应用介绍**: 简要描述项目用途

### 3. 添加Key

1. 在应用详情页点击「添加Key」
2. 选择产品类型：
   - ✅ **JavaScript API GL**（必选，用于前端地图展示）
   - ✅ **WebService API**（可选，用于服务端调用）
3. 填写Key名称，勾选所需API服务：
   - 地图显示
   - POI搜索
   - 路线规划
   - 地理/逆地理编码
   - 定位服务
4. 提交创建

### 4. 配额说明

| API类型 | 免费配额 | 适用场景 |
|---------|---------|----------|
| 地图加载 | 100万次/天 | 前端地图展示 |
| POI搜索 | 5000次/天 | 地点搜索 |
| 路线规划 | 1000次/天 | 导航规划 |
| 地理编码 | 5000次/天 | 地址转坐标 |

**免费配额完全足够本Demo使用！**

### 5. 获取密钥

创建成功后，在控制台可以看到生成的Key（格式：`XXXX-XXXX-XXXX-XXXX-XXXX-XXXX`）

复制该Key，在项目中配置：
```html
<script src="https://map.qq.com/api/gljs?v=1.exp&key=YOUR_KEY"></script>
```

或在应用界面输入框中粘贴保存。

## 四、常见问题

### Q1: 地图无法显示？

**检查项**:
- API Key是否正确配置
- 是否勾选了「JavaScript API GL」服务
- 浏览器控制台是否有错误提示
- 网络是否可以访问腾讯地图服务器

**解决方法**:
- 确认Key格式正确（32位字符）
- 在控制台检查配额是否充足
- 检查浏览器开发者工具（F12）Console面板

---

### Q2: POI搜索无结果？

**可能原因**:
- 未开通WebService API服务
- 搜索关键词过于模糊
- 地区参数不正确

**解决方法**:
- 在控制台添加WebService API Key
- 尝试更明确的搜索词（如「咖啡」而非「好吃的」）
- 明确指定地区（如「北京」而非「附近」）

---

### Q3: 定位功能不工作？

**原因**:
- 浏览器未授权位置权限
- 使用HTTP协议（非HTTPS）某些浏览器限制定位

**解决方法**:
- 点击浏览器地址栏左侧，允许位置访问
- 使用HTTPS部署（本地测试用localhost可正常工作）

---

### Q4: AI对话功能报错？

**如果是本地规则引擎**:
- 无需API Key，默认可用
- 可能是输入格式无法识别，尝试更明确的表达

**如果是云端AI API**:
- 检查API Key是否正确
- 检查配额是否充足
- 检查网络连接

## 五、安全提示

⚠️ **重要安全建议**:

1. **不要公开泄露API Key**
   - GitHub公开仓库不要硬编码Key
   - 使用环境变量或配置文件管理

2. **配额监控**
   - 定期检查控制台配额使用情况
   - 设置配额预警

3. **生产环境建议**
   - 使用服务端代理API调用
   - 实现IP白名单限制
   - 添加请求频率限制
