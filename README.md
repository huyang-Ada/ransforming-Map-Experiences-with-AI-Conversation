# AI智能地图助手

> 🤖 用自然语言与地图对话，让搜索变得更简单
> 
> 基于腾讯位置服务 + AI大模型的智能地图应用

---

## 📖 项目简介

**AI智能地图助手**是一个创新的地图交互应用，用户无需学习复杂的地图操作，只需用自然语言描述需求（如"找北京西站附近的咖啡馆"、"从天安门到故宫怎么走"），AI自动解析意图并调用地图API，返回精准结果。

**核心特性**：

- ✅ **自然语言交互** - 口语表达即可，无需专业操作
- ✅ **智能意图识别** - 自动判断搜索、导航、推荐等意图
- ✅ **混合AI架构** - 本地规则引擎 + 云端大模型灵活切换
- ✅ **地图可视化** - 实时标记、路线绘制、详情弹窗
- ✅ **零门槛使用** - 免费API额度充足，无需后端服务器

## 🚀 快速开始

### 前置要求

- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 腾讯位置服务API密钥（免费申请）
- AI大模型API密钥（可选，不配置也可使用本地引擎）

---

### 步骤1：申请API密钥

#### 腾讯地图API密钥（必需）

1. 访问 [腾讯位置服务官网](https://lbs.qq.com/)
2. 注册账号并登录
3. 进入控制台创建应用，勾选「JavaScript API GL」和「WebService API」
4. 获取API Key（格式：`XXXX-XXXX-XXXX-XXXX-XXXX-XXXX`）

**免费配额**：
- 地图加载：100万次/天
- POI搜索：5000次/天
- 路线规划：1000次/天

详见：[docs/API申请指南.md](docs/API申请指南.md)

---

#### AI大模型API密钥（可选）

支持以下AI引擎（不配置也可使用本地规则引擎）：

| AI引擎 | 免费额度 | 申请地址 |
|--------|---------|----------|
| 通义千问 | 100万tokens | [阿里云DashScope](https://dashscope.console.aliyun.com/) |
| 智谱GLM | 100万tokens | [智谱AI开放平台](https://open.bigmodel.cn/) |
| 自定义API | 自行配置 | 填写URL和Key |

---

### 步骤2：配置项目

#### 方法A：界面配置（推荐）

1. 用浏览器打开 [index.html](index.html)
2. 点击「⚙️ API配置」展开配置面板
3. 输入腾讯地图API Key，点击保存
4. 选择AI引擎（默认「本地规则引擎」即可）
5. 如使用云端AI，输入对应API Key

配置自动保存到浏览器本地存储，下次打开无需重新配置。

---

#### 方法B：代码配置

在 [index.html](index.html) 第6行替换API Key：


## 🔧 技术实现细节

### 核心API使用

#### POI搜索

```javascript
// 1. 创建搜索服务
const searchService = new TMap.search.Search({
    pageSize: 10
});

// 2. 执行搜索
const result = await searchService.search({
    keyword: '咖啡馆',
    region: '北京',
    location: { lat: 39.89, lng: 116.32 } // 中心点坐标
});

// 3. 在地图标记
result.data.forEach(poi => {
    new TMap.Marker({
        position: poi.location,
        map: map
    });
});
```

---

#### 路线规划

```javascript
// 1. 创建路线规划服务
const driving = new TMap.service.Driving({
    policy: TMap.constants.DRIVING_POLICY.LEAST_TIME
});

// 2. 规划路线
const route = await driving.search({
    from: new TMap.LatLng(39.90, 116.40), // 天安门
    to: new TMap.LatLng(39.92, 116.39)    // 故宫
});

// 3. 绘制路线
new TMap.MultiPolyline({
    map: map,
    geometries: [{
        positions: route.result.routes[0].polyline
    }]
});
```

---

### AI意图识别

#### 本地规则引擎

```javascript
// 正则匹配常见句式
const intents = {
    search_poi: [
        /找(.+)附近(.+)/,   // "找北京西站附近的咖啡馆"
        /搜索(.+)/,         // "搜索美食"
        /附近(.+)/          // "附近景点"
    ],
    route_planning: [
        /从(.+)到(.+)/,     // "从天安门到故宫"
        /(.+)到(.+)怎么走/  // "北京西站到故宫怎么走"
    ]
};

// 提取参数
const match = input.match(/找(.+)附近(.+)/);
if (match) {
    return {
        intent: 'search_poi',
        params: {
            location: match[1], // "北京西站"
            keyword: match[2]   // "咖��馆"
        }
    };
}
```
