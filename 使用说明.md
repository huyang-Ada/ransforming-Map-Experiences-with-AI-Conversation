## 📋 第一步：申请API密钥

### 1. 腾讯位置服务API密钥（必需）

**申请步骤**：
1. 访问 https://lbs.qq.com/
2. 注册账号并登录
3. 进入控制台 → 应用管理 → 创建应用
4. 填写应用信息，勾选「JavaScript API GL」和「WebService API」
5. 获取API Key（格式：XXXX-XXXX-XXXX-XXXX）

**免费配额**：
- 地图加载：100万次/天
- POI搜索：5000次/天
- 路线规划：1000次/天
- **完全免费，足够测试使用！**

详细申请指南：[docs/API申请指南.md](docs/API申请指南.md)

---

### 2. AI大模型API密钥（可选）

**不申请也可使用**！项目默认使用本地规则引擎（免费、快速）

如果需要更智能的对话能力，可申请：

- 智谱GLM：https://open.bigmodel.cn/（100万tokens免费）

## 🚀 第二步：配置项目

### 方式A：界面配置（推荐）

1. 用浏览器打开 `ai-chat-map/index.html`
2. 点击页面上的「⚙️ API配置」按钮
3. 输入你的腾讯地图API Key
4. 点击「保存」按钮

配置会自动保存到浏览器，下次打开无需重新输入！

---

### 方式B：代码配置

打开 `ai-chat-map/index.html` 文件，找到第6行：

```html
<script src="https://map.qq.com/api/gljs?v=1.exp&key=YOUR_TENCENT_MAP_KEY"></script>
```

将 `YOUR_TENCENT_MAP_KEY` 替换为你的实际API Key：

```html
<script src="https://map.qq.com/api/gljs?v=1.exp&key=ABCD-EFGH-IJKL-MNOP-QRST-UVWX"></script>
```

保存文件后用浏览器打开即可！

## 🔧 常见问题解决

### 问题1：地图无法显示
**原因**：API Key未配置或错误
**解决**：
- 检查浏览器控制台（F12）是否有错误
- 确认Key格式正确（32位字符）
- 确保已勾选「JavaScript API GL」服务

---

### 问题2：POI搜索无结果
**原因**：未开通WebService API
**解决**：
- 在腾讯控制台添加WebService API Key
- 确认配额未用尽

---

### 问题3：定位功能不工作
**原因**：浏览器未授权位置权限
**解决**：
- 点击地址栏左侧，允许位置访问
- 使用HTTPS或localhost（HTTP可能被浏览器限制）
